# 🏃‍♂️ SportsTalentAI

SportsTalentAI is an AI-powered Android application designed to analyze athletic performance using real-time video recording and intelligent pose analysis.

The application records athlete movements using CameraX, evaluates performance metrics, and generates a structured assessment including score, grade, feedback, and eligibility insights.

This project demonstrates practical implementation of modern Android architecture and on-device performance analysis systems.

---

## 🚀 Features

- 🎥 Real-time video recording using CameraX
- 🤖 Pose-based movement analysis
- 📊 Performance score calculation
- 🏅 Grade generation (A / B / C)
- 🧠 College eligibility evaluation engine
- 💾 Local storage using Room Database
- 📈 Dashboard with performance insights
- 🎨 Material3 professional UI design

---

## 🧱 Architecture

The app follows a clean MVVM architecture pattern:

- **UI Layer** → Activities, Views, Layouts
- **ViewModel Layer** → Business logic & state handling
- **Repository Layer** → Data management
- **Room Database** → Local persistence
- **AI Module** → Pose analysis & scoring engine

---

## 🛠️ Tech Stack

- Kotlin
- Android SDK
- CameraX (Video + Image Analysis)
- Material 3 UI
- Room Database
- MVVM Architecture
- LiveData / ViewModel
- On-device evaluation engine
